## ✅ ADS
- 영화관에서 함께 웃고 떠들며 즐기고 싶은 사람들을 위해
- 응원 등 시끄럽게 볼 수 있는 '영화관 콘텐츠'를 빠르게 접하고, 쉽게 예매할 수 있게 정보를 제공하며
- 직접 대관을 희망하는 사용자를 위해 '파티원 모집' 서비스를 제공

## ✅ 주요기능 및 차별성 💡
- 함께 웃고 떠들며 응원할 수 있는 '영화관 콘텐츠' 정보 제공
- 원하는 콘텐츠의 쉬운 예매 시스템 (신청하기, 공유하기, 신청취소, 커뮤티니 등)
- 직접 대관을 희망하는 사용자의 '파티원 모집' 기능 및 영화관 정보 제공 (모집하기)
  

<br/><br/>

## ✅ 화면 구성 (4개의 탭바로 구성: 홈, 검색, 모집, 마이페이지)
1. 홈
- 영화관 '응원 상영' 콘텐츠 정보를 한 곳에 모아 보여줌
  - KBO, 영화, 콘서트, 애니메이션, 스포츠, 기타(드라마 등)
  - 스포츠, 영화, 콘서트 등 카테고리를 터치하면 해당 콘텐츠 리스트 제공
  - 보고싶은 콘텐츠 터치 시, '상세 정보' 및 '신청하기', '공유하기 기능 제공
  - 상세정보 화면 : [콘텐츠 세부 정보, ‘신청하기’, ‘공유하기’, 모집 현황(성별, 연령별), 좋아요, 댓글 기능] <br/>

  - 검색창 터치 시, ‘최근 검색기록’과 ‘검색기록 지우기’ 기능 제공 <br/>

  - ‘모집하기’ 화면 : 사진, 카테고리, 제목, 모집인원, 장소, 일시, 해시태그 기능 <br/>

## ✅ 피드백 / 개선사항
### 1. 조별 피드백
- “앱에 대한 설명이 있었으면 좋겠다” <br/>
  ➡️  로그인 화면에 ‘앱 설명’, ‘사용방법’ 등 소개 <br/><br/>
  
- “프로필에 ‘연령대’가 나왔으면 좋겠다” <br/>
  ➡️  사용자 프로필, 신청자 등에 ‘연령대’ 추가 <br/><br/>
  
- “신청 후 취소 기능이 있었으면 좋겠다” <br/>
  ➡️  신청 취소 기능 반영 <br/><br/>
  
- ”캘린더 하루에 두 개 이상의 영화 일정이 잡힌 경우가 표현되었으면 좋겠다” <br/>
  ➡️  캘린더 내 숫자를 통해 일정 개수 표현 및 클릭 시 상세 일정 페이지 이동 추가 <br/><br/>

<br/>

### 2. 강사님 피드백
- 첫 화면(로그인 화면)<br/>
  - "앱 소개 장면이 넘어가는 속도가 빠름. 로그인 기능의 반영 시점 변화 (신청하기 또는 모집하기 기능 사용 시 '로그인' 기능 반영 등)"<br/><br/>

- 홈 화면<br/>
  - "카테고리와 콘텐츠의 줄간격 수정 (올바른 그룹핑)", "부등호(>) 표시는 '심볼'로 교체"<br/><br/>

- 상세화면<br/>
  - "'상세설명', '모집현황' 부분은 bold를 써서 '타이틀' 같은 느낌을 줄 것. 본문 텍스트보다 작으면 안됨."<br/>
  - "성별 현황', '연령별 현황' 텍트그 자간은 줄일 것" <br/>
  - "'신청하기' 버튼에서 'Alert' 기능은 불필요." <br/><br/>

- 댓글창 <br/>
  - "아이디와 본문의 구분이 필요함 (텍스트 크기와 굵기가 비슷함. 차이를 줄 것)" <br/><br/>

- 검색창 <br/>
  - "'돋보기'와 'x' 심볼의 좌우 여백 필요" <br/><br/>

- 모집 <br/>
  - "플로팅 버튼('+모집' 버튼)의 위치 수정 (기존: 하단 -> 수정: 상단 내비게이션바 우측)" <br/><br/>
